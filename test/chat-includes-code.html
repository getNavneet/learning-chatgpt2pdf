To convert text to PDF in Node.js, several libraries can help you create high-quality PDF documents. Here are the most popular and reliable ones:

✅ 1. pdfkit

Best for: Generating PDFs from scratch with full control (e.g., headers, paragraphs, fonts, layout)

Installation: 

bash
npm install pdfkit

Example: 

js
const PDFDocument = require('pdfkit');
const fs = require('fs');

const doc = new PDFDocument();
doc.pipe(fs.createWriteStream('output.pdf'));

doc.fontSize(12).text('Hello, this is a simple text-to-PDF conversion using PDFKit!', 100, 100);

doc.end();

✅ 2. html-pdf

Best for: Converting HTML strings to PDF (uses PhantomJS under the hood)

Installation: 

bash
npm install html-pdf

Example: 

js
const pdf = require('html-pdf');

const html = '<h1>Hello World</h1><p>This is a PDF generated from HTML</p>';
pdf.create(html).toFile('output.pdf', function(err, res) {
  if (err) return console.log(err);
  console.log(res); // { filename: '/path/to/output.pdf' }
});

⚠️ html-pdf is no longer actively maintained. You may consider using puppeteer instead.

✅ 3. puppeteer

Best for: Headless Chrome-based HTML-to-PDF rendering (modern and powerful)

Installation: 

bash
npm install puppeteer

Example: 

js
const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  await page.setContent('<h1>Hello World</h1><p>This is a PDF generated by Puppeteer</p>');
  await page.pdf({ path: 'output.pdf', format: 'A4' });

  await browser.close();
})();

✅ 4. jspdf

Best for: Client-side PDF generation in the browser (can also be used in Node with polyfills)

Installation: 

bash
npm install jspdf

Example: 

js
const { jsPDF } = require('jspdf');
const fs = require('fs');

const doc = new jsPDF();
doc.text("Hello world!", 10, 10);
fs.writeFileSync('output.pdf', Buffer.from(doc.output('arraybuffer')));

Recommendation

For pure text-to-PDF use in Node.js, go with:

  • ✅ pdfkit – if you want full control and don’t need to render HTML.

  • ✅ puppeteer – if you’re rendering styled HTML to PDF and want high-quality output.

🚀 Full Example: Modify Live Website → PDF 

js

                                  
                                  
const puppeteer = require('puppeteer');

(async () => {
  const browser = await puppeteer.launch();
  const page = await browser.newPage();

  // 1. Navigate to the target website
  await page.goto('https://example.com', { waitUntil: 'networkidle0' });

  // 2. Inject your custom CSS and DOM changes
  await page.addStyleTag({
    content: `
      body { font-family: 'Arial'; background-color: #f9f9f9; }
      h1 { color: darkred; }
      .my-banner { padding: 20px; background: #ffd; text-align: center; font-weight: bold; }
    `
  });

  await page.evaluate(() => {
    // Example DOM modification: Add a banner at the top
    const banner = document.createElement('div');
    banner.className = 'my-banner';
    banner.innerText = '🚀 This PDF is modified by Navneet!';
    document.body.prepend(banner);

    // Change the title text
    const h1 = document.querySelector('h1');
    if (h1) h1.innerText = 'Modified by Puppeteer Bot ✨';
  });

  // 3. Generate and save PDF
  await page.pdf({
    path: 'modified-website.pdf',
    format: 'A4',
    printBackground: true
  });

  await browser.close();
})();
